ALTER - Вносит изменения в БД или конкретную таблицу
CREATE - Создание БД   (CREATE DATABASE test;) 
DELETE - Удаление строки с БД
DROP - Удаляет БД
EXIT - Выход
INSERT - Вставка или размещение данных (запись)
RENAME - Переименование
SOURCE - Выполнение команд из файла
STATUS - Отображает текущий статус
UPDATE - Обновление данных
USE - Использование БД
